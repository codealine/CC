Detailed Description

Objects for each gift basket exchanged between couples.

Contains information about total cost of gifts, total value of gifts, value of luxury items. Contains arrays holding objects of each type of gift present in the gift basket, namely Essential, Utility, and Luxury.

Also contains method to print information of all the gift exchanges and log them to a file
